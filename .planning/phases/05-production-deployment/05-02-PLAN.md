  <action>
Create a GitHub Actions workflow for automated deployment to Azure Container Apps.

Workflow structure:

```yaml
name: Deploy to Azure

on:
  push:
    branches: [main]

env:
  RESOURCE_GROUP: inclusify-rg
  ACR_NAME: inclusifyacr
  CONTAINERAPPS_ENV: inclusify-env
  BACKEND_APP: inclusify-backend
  FRONTEND_APP: inclusify-frontend

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Log in to Azure
        uses: azure/login@v1
        with:
          creds: ${{ secrets.AZURE_CREDENTIALS }}

      - name: Log in to ACR
        run: az acr login --name ${{ env.ACR_NAME }}

      - name: Build and push Backend
        run: |
          docker build \
            -f infra/docker/backend.Dockerfile \
            --target runtime \
            -t ${{ env.ACR_NAME }}.azurecr.io/inclusify-backend:${{ github.sha }} \
            .
          docker push ${{ env.ACR_NAME }}.azurecr.io/inclusify-backend:${{ github.sha }}

      - name: Build and push Frontend
        run: |
          docker build \
            -f infra/docker/frontend.Dockerfile \
            --target runner \
            -t ${{ env.ACR_NAME }}.azurecr.io/inclusify-frontend:${{ github.sha }} \
            .
          docker push ${{ env.ACR_NAME }}.azurecr.io/inclusify-frontend:${{ github.sha }}

      - name: Deploy Backend
        run: |
          az containerapp update \
            --name ${{ env.BACKEND_APP }} \
            --resource-group ${{ env.RESOURCE_GROUP }} \
            --image ${{ env.ACR_NAME }}.azurecr.io/inclusify-backend:${{ github.sha }}

      - name: Deploy Frontend
        run: |
          az containerapp update \
            --name ${{ env.FRONTEND_APP }} \
            --resource-group ${{ env.RESOURCE_GROUP }} \
            --image ${{ env.ACR_NAME }}.azurecr.io/inclusify-frontend:${{ github.sha }}

      - name: Verify deployment
        run: |
          BACKEND_URL=$(az containerapp show \
            --name ${{ env.BACKEND_APP }} \
            --resource-group ${{ env.RESOURCE_GROUP }} \
            --query "properties.configuration.ingress.fqdn" -o tsv)
          echo "Backend URL: https://$BACKEND_URL"
          # Wait for deployment to stabilize
          sleep 30
          # Health check
          curl -sf "https://$BACKEND_URL/health" || exit 1
          echo "Backend health check passed"
```

Key decisions:
- Trigger: push to main only (per CONTEXT.md)
- Uses git SHA for image tags (avoids caching issues)
- No --force flags, no destructive operations
- Health check verification at end
- Uses az containerapp update (not create) since 05-01 creates the apps
