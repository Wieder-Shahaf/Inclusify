   - Import Issue from router (avoid circular by importing type only)

   - Import CircuitBreakerError from circuit_breaker

   - Function calculate_overlap(issue1: Issue, issue2: Issue) -> float:

     - Return overlap_chars / min(len1, len2)

   - Function merge_results(llm_issues: list[Issue], rule_issues: list[Issue], overlap_threshold: float = 0.5) -> list[Issue]:
     - Start with all LLM issues
     - For each rule issue, add only if no LLM issue overlaps > threshold
     - Sort by start position

   - Class HybridDetector:
     - __init__(self, vllm_client: VLLMClient = None):
       - Create default client if not provided
     - async def analyze(self, text: str, language: str = 'auto') -> tuple[list[Issue], str]:
       - Detect language if 'auto' (simple heuristic: Hebrew chars present)
       - Split text into sentences with offsets
       - For each sentence:
         - Try LLM analysis
         - If success and severity not "Correct", create Issue
         - If failure, mark sentence for fallback
       - Run rule-based on full text
       - Merge LLM and rule results
       - Determine mode: all_llm -> "llm", some_llm -> "hybrid", no_llm -> "rules_only"
       - Return (merged_issues, mode)
