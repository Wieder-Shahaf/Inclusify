Claude creates systemd unit file for vLLM server:

- Service type: simple
- User: azureuser
- WorkingDirectory: /home/azureuser/inclusify
- Environment: CUDA_VISIBLE_DEVICES=0
- ExecStart: vllm serve with these flags (per CONTEXT.md decisions):
  - Model path: /home/azureuser/models/suzume-llama-3-8B-gptq
  - --dtype float16 (T4 supports float16, not bfloat16)
  - --max-model-len 2048 (reduced from default 4096 to save VRAM)
  - --port 8001
  - --host 127.0.0.1 (localhost only, no public access)
  - --enable-lora
  - --lora-modules inclusify=/home/azureuser/inclusify/ml/LoRA_Adapters
  - --gpu-memory-utilization 0.9
- Restart: always
- RestartSec: 10

Include [Unit] section with After=network.target and [Install] section with WantedBy=multi-user.target.
