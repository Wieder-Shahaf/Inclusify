## User Setup Required

**External services require manual configuration.** Before OAuth can work:

1. **Google Cloud Console Setup:**
   - Create OAuth 2.0 credentials at https://console.cloud.google.com/apis/credentials
   - Set Authorized redirect URI to: `http://localhost:8000/auth/google/callback` (dev) or production callback URL
   - Copy Client ID and Client Secret

2. **Environment Variables:**
   ```bash
   export GOOGLE_CLIENT_ID="your-client-id.apps.googleusercontent.com"
   export GOOGLE_CLIENT_SECRET="your-client-secret"
   export FRONTEND_URL="http://localhost:3000"  # or production URL
   ```

3. **Verification:**
   ```bash
   # Start backend
   cd backend && uvicorn app.main:app --reload --port 8000

   # Visit authorize endpoint - should redirect to Google
   open http://localhost:8000/auth/google/authorize
   ```
