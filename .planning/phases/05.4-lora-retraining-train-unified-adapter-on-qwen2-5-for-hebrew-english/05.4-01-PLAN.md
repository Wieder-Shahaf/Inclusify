**ml/training/prepare_data.py:**
Implement 4 functions:

1. `load_dataset(csv_path: str) -> pd.DataFrame`:
   - Load augmented_dataset.csv with pandas
   - Validate required columns: "Sentence", "Severity Label", "Explanation"
   - Return DataFrame

2. `stratified_split(df: pd.DataFrame, test_size: float = 0.2, random_state: int = 42) -> Tuple[pd.DataFrame, pd.DataFrame]`:
   - Use sklearn.train_test_split with stratify=df["Severity Label"]
   - Return (train_df, val_df)
   - Preserve class proportions in validation set

3. `format_example_qwen(sentence: str, severity: str, explanation: str, tokenizer) -> str`:
   - Create messages list with system/user/assistant roles
   - System: Use Inclusify-specific prompt (classify sentences by severity: Correct, Outdated, Biased, Potentially Offensive, Factually Incorrect)
   - User: sentence
   - Assistant: JSON with severity and explanation
   - Apply tokenizer.apply_chat_template(messages, tokenize=False, add_generation_prompt=False)
   - Return formatted string

4. `prepare_datasets(csv_path: str, tokenizer, test_size: float = 0.2) -> Tuple[Dataset, Dataset]`:
   - Load dataset
   - Stratified split
   - Format each example with Qwen2.5 chat template
   - Convert to HuggingFace Dataset objects
   - Return (train_dataset, val_dataset)

**Dependencies:**
```python
import pandas as pd
from sklearn.model_selection import train_test_split
from datasets import Dataset
from typing import Tuple
```

**Why this approach:**
- tokenizer.apply_chat_template() prevents special token errors (not manual string concatenation)
- Stratified split critical for small dataset (1001 samples) with class imbalance
- HuggingFace Dataset format integrates with SFTTrainer
