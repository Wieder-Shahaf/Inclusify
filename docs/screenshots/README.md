# Screenshots

This directory contains screenshots for the README and documentation.

## Required Screenshots

To complete the README, please add the following screenshots:

1. **upload.png** - Upload interface showing drag-and-drop functionality
2. **analysis.png** - Analysis results page with highlighted text
3. **details.png** - Issue detail panel showing explanation and alternatives
4. **report.png** - Export/download report interface

## Screenshot Guidelines

- **Resolution**: 1920x1080 or 2560x1440 (retina)
- **Format**: PNG with transparency where applicable
- **Content**: Use realistic demo data (no personal information)
- **UI State**: Show the interface in use (not empty states)
- **Browser**: Use a clean browser window (hide bookmarks bar)
- **Theme**: Capture in light mode (primary theme)

## Naming Convention

- Use lowercase with underscores
- Be descriptive but concise
- Include state if relevant (e.g., `modal_open.png`)
